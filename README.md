# Polish Grammar — B1 (Poprawność gramatyczna)

A tiny self-contained practice app for the *POPRAWNOŚĆ GRAMATYCZNA* module
(exercises V and VI) from the past B1 state exam sheets (certyfikatpolski.pl).

- **V** — ask about the underlined part of the sentence (type the question word).
- **VI** — rewrite the sentence using the word in brackets, keeping the meaning.

Open `index.html` locally, or use the published version on GitHub Pages.

The `*.md` files contain the same exercises in printable form.
